## Project Management

This project uses the monorepo manager [Rush](https://rushjs.io).

- [Project Structure](./docs/project_structure.md)

## Usage

### Development

```
rush update
```

```
rush build
```

```
rush test
```


### Deployment

```
rush prepare-cloud-deployment
```

```
rush deploy-to-cloud
```
